## Estrutura do Projeto
* `data/`: Contém os sinais (.edf). Bloqueado no .gitignore por ser pesado.
* `src/`: Scripts de processamento e modelos.
* `notebooks/`: Exploração e visualização dos sinais.

## Como rodar
1. Crie o ambiente: `py -m venv .venv`
2. Ative: `.\.venv\Scripts\activate`
3. Instale as dependências: `pip install -r requirements.txt`
